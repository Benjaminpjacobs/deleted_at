# DeletedAt

Provides functionality to ActiveRecord::Base to delete records while keeping them in the database.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'deleted_at'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install deleted_at
